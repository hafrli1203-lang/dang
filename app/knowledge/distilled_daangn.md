## 당근 매체 본질
- 경험형·발견형 매체. 정보 학습용 X, 목적형 쇼핑 X. "동네 어슬렁" 심리.
- 이용자 페르소나: "약간의 리스크 감수하더라도 가성비가 크면 움직임". 가격대비가치 민감, 선전·선동(텍스트 퍼포먼스)에 빠르게 반응.
- 핵심 키워드: **싸게 · 가성비 · 충동**.
- 피드 환경: 투박한 일상 사진 다수 → **상대성 원리**. 평범 피드 속 "조금만 신경 쓴 이미지"가 시선 독점.
- 외부 이탈 기피: 당근 유저는 외부 링크 싫어함 → 비즈프로필 내부 전환 우선.

## 성공 광고 2공식
1. **정체성 매칭**: 내 상품 × 당근 유저 교집합. 예) "점심 됩니다"(X) → "가족과 즐기는 푸짐한 점심특선 50% 할인"(O).
2. **충동 생성**: 파격 할인 / 압도적 이미지 / 희소성 / 명분 부여(지각된 가치).
- 관여도 낮은 저가·고할인율 상품 유리. 그 외엔 명분·희소성·이미지로 충동 설계.

## 비즈프로필 = 무료 노출 자산
| 기능 | 노출 지면 | 핵심 역할 |
|---|---|---|
| 소식 | 단골 메인 피드, 동네업체소식 검색 | 상세페이지 역할 |
| 쿠폰 | 동네지도-쿠폰북, 소식 상단 뱃지 | 클릭률 폭증 미끼 |
| 후기 | 추천업체 후기, 카테고리 후기 | 전환율 |
| 상품 | 추천가게 상품 피드 | 가격·구성 사전 노출 |
| 단골 | 단골 피드 무료 노출 | 영구 무료 방송국 |

## 비즈프로필 세팅 체크리스트
- 프로필: 간판/로고 X, 대표 메뉴·시술 사진 O.
- 소개: 인사 X, 특징·철학 1줄 (예: "2대째 비법 육수", "문제성 피부 전문").
- 필수 노출 3정보: **상품 / 가격 / 구매방법(위치·영업시간·포장·배달)**.
- 자동응답: 가격·위치·주차·이용방법·배달여부 사전등록. 응대 지연 시 사기로 단정 → 이탈.
- 단골 CTA: 소식 하단 "단골 맺고 소식 받기" 고정.
- 점검: 체크 10+ 광고 즉시 가능 / 6~9 보완 / 5↓ 광고 금지.

## 소식(랜딩) 3유형
- **Type A 진정성**: 사장·재료·실패·집착 스토리. 상업성 제거.
- **Type B 긴급성**: "오늘만", "선착순 10팀", "재료 소진 시 마감" + 싸게 파는 **명분**(오픈기념·가족의 달) 필수, 의심 제거.
- **Type C 가성비**: 구성을 숫자로. "배달앱 3만원 구성 → 홀 1만원". 박리다매 철학 + 현장반응 묘사.

## 쿠폰 설계 원칙
- CTR **1% → 5%** 가능. 가게 정체성 + 고객 체감혜택 연결.
- 형식: 기간 한정 / 금액 할인 / 서비스 증정(직관성 ↑, 예: 1,000원 할인보다 "음료 무료").
- 신규 소식엔 반드시 쿠폰 첨부 → [쿠폰] 뱃지 동시 노출.

## 광고 모드: 전문가 모드 고정
- 간편모드(자동입찰) = 예산 소진 우선 → CPC 폭등. 실측 CPC 780원 / CPA 41,861원 / 문의 3건.
- 전문가모드(수동입찰) = 클릭 상한 통제. 실측 CPC 240원 / CPA 9,284원 / 문의 20건. **효율 4배+**.
- 입찰: **수동 + 일반게재** 기본. 자동입찰·빠른게재는 예산 초과 위험.

## 광고 구조 & 명명
- 캠페인(목적) → 광고그룹(타겟·예산·입찰) → 소재(이미지+텍스트 30자).
- 기본 구조: **1캠페인 - 1그룹 - 1소재** (테스트용, 소재 묻힘 방지).
- 예산 소진 안 될 때만: 1그룹에 같은 메시지 소재 묶기 / 동일 소재 다타겟.
- 캠페인명 규칙: `목적_지역_나이성별_콘텐츠유형` (예: `비프홍보_강남_30F_영상A`).

## 소재 운영
- 같은 이미지에 텍스트만 N종 → 클릭율 상위만 ON, 나머지 OFF.
- 텍스트 30자 이내, 자극적·구체 숫자·문장부호("",(),...,?!) 1개 이상.
- 키워드 광고는 메인키워드만 등록(검색량 낮음).
- 외부 전환 캠페인·카탈로그는 **저관여 상품**에서만 유리.

## CTR 기준 (이미지 광고)
- < 1% : 소재 교체 필요(관심 없음).
- ≥ 1% : 반응 시작 → 예산 유지.
- ≥ 2~3% : 대박 → 예산 증액·노출 극대화.
- 업종·객단가별 편차 큼. 추적 가능하면 **CPA 우선**.

## 고객 행동 계단 (관심도 비용화)
1. 클릭 ★ / 2. 관심있어요 ★★ / 3. 단골 ★★★ / 4. 쿠폰다운 ★★★★ / 5. 채팅문의 ★★★★★.
- 각 단계 **단가화**(클릭당/관심당/단골당/쿠폰당/문의당 원).
- 상품별 진짜 매출 견인 지표 1개 식별 → 그 단가 낮추는 데 집중.

## 절대 공식: 고객획득비용 ≤ 판매이익
- 깨지면 적자.
- 테스트 예산 = 마진 **3~5배수**.
- 예: 마진 1만원 → 3~5만원 집행.
- 전환 발생: CAC를 마진 이하로 낮추는 고도화 단계.
- 5만원 무전환: 즉시 폐기 → 소재 신규제작 또는 상품 매력도(가격·구성) 재점검.

## 미끼-그물 매칭
- 소식 1 : 소재 5 (자극·감성·가성비·진정성·긴급 버전) → 반응 좋은 조합만 예산 집중.
- 동일 그물(소식)에 다양한 미끼(소재) 테스트가 표준.

## 최적화 사이클: 무료노출 → 유료노출 → 측정 → 최적화
- 무료 기반(비즈프로필) 부실 시 유료 효율 폭락.
- 측정: 클릭·관심·단골·쿠폰·문의·**댓글·후기**까지 비용화. 댓글·후기 폭증일에 매출↑면 KPI = 댓글당 비용.
- 최적화: A광고(클릭 미끼) + B광고(전환 그물) → **A의 이미지 + B의 본문 + 댓글에 등장한 표현** = 통합 소재.

## 콘텐츠 카피 9종 프레임 (15~25자, 인사·형용사 금지)
- **소셜프루프**: (실제)·(후기)·(인증)·속보·간증 포함 + [효능/결과]. 예: "살이 진짜 빠진다" 리뷰 속출.
- **호기심·결핍**: 이거·비밀·단 하나·돈·운·재물·해결·종결. 대명사로 정체 은폐. 예: 냄새 고민, '이거' 하나면.
- **권위·구체성**: 출처·고유명사·수치. 예: 청와대 납품 그 떡 3대째 / 수율 98% 박스.
- 금지어: 최고의·선물같은·정성스런·행복한.
- 필수: ""·()·…·?! 중 1개 이상.

## AI 콘텐츠 생성 입력 표준
- 입력: 상세페이지 캡쳐 + 기본정보 + (벤치마킹 경쟁사 콘텐츠 구조).
- 경쟁사 콘텐츠 활용 = **논리구조만 추출, 톤·내용은 자사로 교체**.
- 화자: 1인칭 사장님, "~해요/~더라고요" 구어체, 광고 말투 금지("강력추천","품절임박","대박" X).
- 이미지 디테일은 "발견·경험"으로 서술("성분표 보니 ~ 들어있더라").

## 후기 관리 4단계
1. 발생: 부정 후기 즉시 인지(좋아요 쌓이면 '유용한 순' 상단 고정).
2. 신고: 점3개 → 신고 → 명예훼손/사생활침해/업무방해 → 사업자등록증(또는 신분증+통신가입내역). 즉시 블라인드.
3. 방어(30일 골든타임): 신규 호평 후기 10+ 확보 → 최신순으로 밀어내기. 블라인드 기간엔 좋아요 누적 불가 → 복구돼도 상단 진입 불가.
4. 종결: 이의제기 없으면 영구 삭제.
