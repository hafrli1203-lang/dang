## 광고 본질 원칙
- 콘텐츠 × 운영 = 성과. 콘텐츠 80%, 운영 20%. 콘텐츠가 0이면 무엇을 곱해도 0
- 판단 기준은 ROAS 아닌 **공헌이익(매출−변동비)** 절대값. ROAS 1000%·일 10만 < ROAS 300%·일 100만
- 당근은 충동구매 플랫폼 → "콘텐츠로 타겟팅"(키워드 아님). 알고리즘보다 **사람의 결핍·행동양식**에 집중

## 5단계 운영 사이클
1. **선택지 늘리기**: 소구별 다양한 소재 발행. 단일 올인 금지
2. **평가하기**: 절대평가(MAX CPA 손익분기) + 상대평가(타 소재 비교)
3. **집중하기**: 저성과 OFF → 고성과로 예산 집중 → 학습 빠른 안정화
4. **보강하기**: 실패·성공 요소 분리 → 신규 소재 수혈(피로도 대비)
5. **키우기**: 검증된 위너만 스케일업

## 캠페인 3계층 사고
| 단계 | 결정 |
|---|---|
| 캠페인 | 목적(전환·트래픽·참여·인지) |
| 광고세트 | 타겟·예산·노출위치 |
| 광고 | 이미지·문구·링크 |

- 콘텐츠 성격 ≠ 캠페인 목적이면 성과 안 나옴
- 오프라인 매장(음식점·미용실·헬스장·병원)은 **참여 캠페인** 우선. 좋아요·댓글·저장·공유·DM이 쌓이는 구조 설계 → 학습 환경 강제 구축. 외부 플레이스 링크는 뎁스만 늘려 비효율

## MAX CPA 예산 프레임
- **MAX CPA = 판매가 − (원가+배송+포장)** = 손익분기 전환비용
- 일예산: 타이트=MAX CPA×1배 / 여유=×2~3배
- 학습 50건/7일 도달 목표 시: MAX CPA × 50 ÷ 7 = 권장 일예산
- 객단가 ↑ 위해 **번들링 먼저** → MAX CPA 자체가 올라 운영 여유 확보

## ON/OFF 매트릭스
| 절대(MAX CPA 이하) | 상대(타 소재 대비) | 처방 |
|---|---|---|
| ✅ | ✅ | 유지·증액 검토 |
| ✅ | ❌ | 유지·우선순위↓ |
| ❌ | ✅ | 추세 더 보기 |
| ❌ | ❌ | OFF |

## 추세 기반 소거법 (3·5·7일)
- 하루 단위 판단 금지 → 3일→5일→7일 **합산 + 날짜별 방향** 동시
- 7일 합산 MAX CPA 초과해도 날짜별 CPA 하향 추세면 → 유지
- 7일 합산 이하여도 날짜별 상승 추세면 → 경계
- 7일 횡보·상승 + 합산 초과 → OFF

## 소재의 역할 (개별 ROAS 함정)
- **인지 역할**: CTR↑·전환↓·저장/공유↑·재생율↑ → 타 소재 전환 견인
- **전환 역할**: 직접 구매 견인
- 인지 의심 소재는 하나씩 끄며 타 소재 변화 관찰
- 콘텐츠 배합(**후킹 : 설득 : 근거**)으로 역할 설계

## 4지표 진단
| CPM | CPC | CVR | ROAS | 처방 |
|---|---|---|---|---|
| ↑ | ↓ | ↑ | ↑ | 유지 |
| ↓ | ↓ | ↓ | ↓ | 콘텐츠·타겟 재점검 |
| ↓ | ↑ | ↓ | ↓ | 콘텐츠 교체 |

- CPM 1~1.5배 상승: 시즌·피로·품질 → 신규 소재
- CPM 2배↑ 급등: 계정 손상·외부 이슈 → 연령·카테고리 조정

## 성과 진단 순서 (절대 어기지 말 것)
1. ROAS → 2. CPM → 3. **모수의 질**(소재가 데려온 사람) → 4. 상세페이지 → 5. **가격(번들링 먼저)**

> 전환율 낮다고 상세페이지부터 고치면 실패. 소재가 데려오는 모수 질이 먼저

## 3단계 발굴·검증·확장
- **1-1-1** (1캠·1세트·1소재): 가능성 테스트. MAX CPA×1~3배 안 전환 1건 + 과정지표(랜딩·장바구니) 신호
- **ABO** (1캠·1세트·다소재): 가능성 통과 소재만, **같은 소구·같은 타겟**끼리 묶기. 다른 소구·타겟 섞으면 비교 불가
- **CBO** (1캠·다세트·다소재): 검증 위너 확장. CPA·ROAS·객단가만 본다. 신규 미검증 소재 절대 투입 금지

## 증액 안전장치
- 잘 돌아가는 광고는 건드리지 말고 **복제**해 별도 예산 (20% 점진증액도 학습 흔듬)
- 복제 배수: 충동구매 2~3배 / 고관여 1~2배
- 결과당 비용 목표: 기존 CPA × 1.2~1.3
- **1일/7일 기여 차이 20% 이내** = 즉시 전환형 = 증액 적합
- 차이 20% 초과 = 인지 역할 → 설득·근거 비중 늘리거나 중간 경로 보강

## 콘텐츠 기획 5단계
1. **패턴 관찰**: 욕구(흐릿)/필요(구체)/수요(현실조건) 구분. 자사·경쟁사 리뷰 마이닝 + 커뮤니티 어휘 수집
2. **타겟 정의**: 가치관·세계관·행동양식·상황 우선. 인구통계는 맥락 보조. "20대 여자" 금지 → "○○ 때문에 ○○하지 못한 사람"
3. **해결책 나열**: 스펙 아닌 "내 생활에서 바뀌는 결과"
4. **고객 언어 치환**: "특허 3중 필터" → "수돗물 바로 마셔도 되는 물"
5. **고도화**: 반응 요소 분리 + "왜 먹혔는지" 원리 해석

## 시선을 멈추는 카피 공식
- 첫 줄 = **타겟 호명 + 트리거 상황**: "~할 때 / ~하다가 / ~하는 순간"
- 메인–서브 인과관계 필수: 문제 던지면 해결 / 결과 선언하면 근거 / 질문하면 답
- 추상심리(불안·걱정) 금지 → 구체 장면
- 공급자 언어(스펙·기술) 금지 → 결과 언어

## 당근 매장 광고 추가 원칙
- 비주얼 충동형(음식·카페): 김·치즈·인테리어 등 감각 자극 + "○○ 동네 ○○ 코스" 등 타겟 상황 결합
- 고관여형(병원·학원·시술): 트리거 상황 + 메인/서브 인과 설계 필수
- 광고 티 나는 포맷 회피 → 커뮤니티 글·후기·메모장 UI 등 네이티브 포맷 활용
- CTA는 한 가지 행동만 지시 (저장/댓글/방문/문의 중 택1)

## 절대 금지 실수
- 단계 생략·한 세트에 미검증 소재 다수 투입
- 하루 데이터로 OFF 결정
- 잘 되는 광고의 문구·타겟·예산 조정
- ABO에 다른 소구·타겟 섞기
- CBO에 신규 미검증 소재 투입
- 전체 CPA만 보고 OFF (성별·연령·지면별 보고서 확인 후)
- 전환율 안 나올 때 상세페이지·가격부터 손대기
- OFF 소재 폐기 (타겟·세트·역할 바꿔 재활용 가능)
