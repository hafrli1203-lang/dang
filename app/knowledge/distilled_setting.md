## 당근 광고 운영 표준 플로우
1. 비즈프로필 세팅 → 2. 쿠폰 발급 → 3. 소식글 작성 → 4. 광고 등록(전문가 모드) → 5. 채팅 응대 판매 → 6. 효율 측정·소재 교체
- 판매는 **반드시 채팅 판매**. 외부 링크/상품 등록 금지(전환 출처 불명, 의사결정 불가).
- 비즈프로필 광고 종류는 **"비즈프로필 알리기"** 고정. 상품판매·앱웹방문은 효율 낮음.
- 광고 모드는 **전문가 모드 고정**, **자동 충전(잔액 5만↓ 시 10만 충전)** 필수 — 중간에 꺼지면 효율 급락.

## 쿠폰 설계 규칙
| 항목 | 설정값 | 이유 |
|---|---|---|
| 분류 | **기타** 선택 | 카피와 동일 워딩 사용 가능 |
| 쿠폰명 | 광고 카피와 **일치하는 키워드** (예: "당도 99% 수박 할인") | 클릭→다운 일관성 |
| 발급 대상 | **단골 전용** | 다운 시 자동 단골 체결 |
| 발급 개수 | **제한 있음, ≤10개** 운영, 소진 시 충전 | 희소성 → 다운율↑ |
| 유효기간 | 기본 1개월, 시즌성 상품은 단축 | — |
| 할인폭 | **1,000원 할인 금지**. 가격을 올리고 **5,000원 할인**처럼 체감 큰 폭으로 | 다운율·문의율 동반 상승 |
- 쿠폰 차감/중복 무시. 채팅에서 "쿠폰 있으시니 할인가로 드립니다"로 응대.

## 소식글 작성 규칙
- 에디터: **사진 모아보기**(빠른 채팅 전환 유리). 블로그형 X.
- 길이: 짧고 간결. **스크롤 depth 깊으면 고민 시간 ↑ → 문의율↓**.
- 필수 포함: ① 광고 카피와 일치하는 본문 ② **쿠폰 임베드** ③ "쿠폰 다운받고 할인 혜택 놓치지 마세요" ④ "채팅으로 문의 주세요" ⑤ 채팅 가능 시간(예: 9–20시).
- 알림 보내기 **항상 ON**. 게시 시간 **09–21시**(외 시간 알림 미발송).
- 광고용 소식글은 **하나로 고정**·수정만(좋아요/댓글/문의 누적 = 줄선집 효과). 재미·정보성 글은 별도로 매일 업로드(단골 알림 부스팅).
- 외부 링크 첨부 금지. 상품 등록 불필요(판매불가 업종 포함).

## 이미지 가이드
- **식품**: 조리 완성·김 모락모락·보글보글 등 **미각 자극 컷**. 생재료·날것·징그러운 컷 금지.
- **스튜디오 광고 티 나는 컷 금지** — "업체스럽다"는 인상 = CTR 저하.
- 당근 톤: **자연스러움·직관성**. AI 이미지도 자연스러워야 함.
- GIF 불가, 정지 이미지만.

## 광고 세팅 표준값
| 항목 | 값 |
|---|---|
| 캠페인 유형 | 비즈프로필 알리기 |
| 캠페인/광고세트 이름 | `MMDD_상품명` (예: `0605_제육`) |
| 타겟 지역 | 전국(배송 불가 지역만 제외) |
| 오디언스 | **기본값 유지, 손대지 않음** (AI 자동 매칭) |
| 1일 예산 | **5만 원** (소재 5개 × 마리당 1만 원 = 학습 최소치) |
| 입찰 방식 | **수동입찰** |
| 기본 입찰가 | **300원** 시작 |
| 광고 일정 | 종료일 없음, 모든 시간 게재 |

### 입찰가 튜닝 룰
- 1일 예산 미소진 → 다음날 **+50원**, 정확히 소진되면 **-10원** 미세조정으로 **적정 입찰가** 탐색.
- 100원대 저입찰 = 학습 지연(예산 소진까지 며칠 → 의사결정 불가). 시간이 자영업자의 비용.

### 소재 운영(A/B 테스트)
- **소재 5개 동시 운영**(임영웅 룰: 모집단 커야 우승자 나옴).
- **1회 1변수 원칙**: 첫 라운드 = **이미지 동일, 제목만 5종**. 승자 제목 확정 후 다음 라운드 = **제목 고정, 이미지 5종**.
- **효율 낮은 소재 끄지 말 것** — 시스템이 자동으로 잘하는 소재에 예산 몰아줌. 끄면 학습 방해.
- 소재 등록 시 "소식 이미지·제목 그대로 사용" **체크 해제** → 광고 제목 별도 입력. **"내 지역 표시" OFF**(전국 광고).
- 소재 이름은 `1`, `2`, `3`… 순번만.

## 성과 벤치마크
| 지표 | 기준 |
|---|---|
| 식품 CTR | **최소 5%, 잘되면 20%까지** |
| 단가 비교 | 당근 300원 ≈ 네이버 8,000원 ≈ 메타 3,000원 수준 (체감 1/10) |
| 채팅 문의→구매 전환 | **거의 100%** (문의수 ≈ 구매수로 간주 가능) |

## 의사결정 지표 — 실제 남는 돈
- **순수익 = 매출 − 원재료 − 광고비**. 이 값이 **+이면 광고 유지·증액**, −이면 카피/이미지/쿠폰 점검.
- 객단가 낮으면 **패키지·세트·무료배송 임계금액**으로 객단가 ↑.
- 마진 1,200원처럼 낮아도 **저단가 상품은 판매량이 커지므로** 광고 효율 가능(모닝 vs 롤스로이스 비유).

## 채팅 응대(말까 제거 원칙)
- 모든 단계에서 고객은 "**~할까 말까**" 갈등 중. **선택지를 좁힌 질문**으로 "말까"를 제거.
  - X: "구매하시겠어요?" → 살까/말까 발생
  - O: "고기만두/김치만두 중 어떤 게 좋으세요?" "몇 분 드세요?" → 살까 확정 후 옵션 선택
- 응대 톤: 기계적 안내문 X. **식당 직원처럼 추천 대화**.
- 긴 안내·주의사항은 **결제 완료 후** 발송. 의사결정 전 장문은 이탈 유발.
- 자주 쓰는 문구 템플릿화 → 복붙 운영으로 품 절감.

## 카테고리·계정 분리 룰
- 한 비즈프로필 = **하나의 카테고리**. 추어탕 계정에서 갈비탕 광고 X.
- 다른 품목·다른 톤 = **비즈프로필 신규 생성**.
- 오프라인 매장 정보는 **온라인 판매 광고에 미포함**(헷갈림 유발 → 전환↓). 지역 표시도 OFF.

## 콘텐츠 운영(발굴→확장 원리)
- 광고 소식글은 **승자 1개 유지·누적**(좋아요/문의/단골 적립이 신뢰 자산).
- 재미·정보성 글은 별도로 **매일 발행** → 단골 알림 발송 → 광고 효과 부스팅.
- 잘된 카피·이미지는 **고정 후 다른 변수만 교체**해 확장.

## 공통 금지 사항
- 외부 링크(스마트스토어 등) 첨부 — 전환 측정 불가.
- 상품 등록 기능 사용 — 채팅 우회.
- 간편 모드 — 기능 제한.
- 자동 충전 미설정 — 광고 끊김 시 학습 리셋.
- 효율 낮은 소재 강제 OFF — 자동 최적화 방해.
- 1,000원 단위 미미한 할인 — 다운율 저조.
- 08시 전/22시 후 소식글 업로드 — 단골 알림 미발송.
