## 클릭 부르는 후크 공식 (실제 문장 그대로)

**1. 의심·반문 후크 — 1순위.** 고객 입에서 나온 그 말을 큰따옴표로 박는다.
- "계산 잘못된 거 아니에요?" 매일 듣습니다
- "렌즈 0원이요?" 매일 되묻는 가격
- "6개월 만에 또 안경도수 올랐어요"
- "이거 진짜 0원이에요?"

"후기 쏟아지는", "문의 폭주 중" 같은 자가발화는 약하다. 의심하는 그 말 자체가 후크다.

**2. 숫자 갭 후크 — 가격 앵커링.**
- 정가 184,000원 → 103,500원
- 보통 20만원 누진안경, 7만원대 끝
- 압축 3번까지 0원, 안경테 39,000원~
- 렌즈 5~8만원 절약, 진해서 여기뿐

원가-할인가 같은 줄. "최저가/가성비" 형용사 금지, 두 숫자만.

**3. 결핍 직격 후크 — 40대+ 페인포인트.**
- 도수 또 올랐다고요? 해결책 있습니다
- 안경값 또 나가기 전에...비밀 하나
- 40대 넘으면 꼭 필요한 '이거'
- 렌즈 추가금 때문에 한숨?

타겟이 매일 하는 그 고민의 문장을 그대로 옮긴다.

**4. "이거"·"비밀"·"종결" 호기심 후크.**
- 안경테만 사면 렌즈는?…이거 실화
- 렌즈값 고민, 이거 하나로 종결됩니다
- 콘택트렌즈 반값의 비밀…3월까지만

답을 후크에 풀지 않는다. 본문에서 반드시 해소(안 그러면 정서적 가치만 주고 끝).

**5. 마감 후크 — 실제 날짜만.**
- 알콘 토탈원 무료증정…3/29 끝입니다
- 3월 29일 지나면 이 가격 없습니다, 11일 남음

"마감임박" 단독·허위 사용 금지(신뢰 즉시 손실). 진짜 마감일 있을 때만.

## 전환되는 소식글 골격 (심곡·진해·공주 공통 패턴)

1. **고객 의심 인용 1줄** — "계산 잘못된 거 아니에요?" / "안경 맞춘 지 6개월밖에 안 됐는데, 벌써 도수가 올랐대요"
2. **공감 1줄** — "부모님 표정이 어두운 걸 저희도 느낍니다" / "한숨 나오셨던 분들 많으실 겁니다"
3. **숫자 등식** — "안경테 79,000원 + 누진렌즈 0원 = 79,000원 (끝)", "90P 2팩 184,000 → 103,500"
4. **장사철학 1줄(신뢰)** — "저는 이 동네에서 오래 장사할 사람입니다. 한 번 맞추고 편하시면 다음에도 오시고 가족분도 데려오시잖아요. 그게 제가 남는 겁니다"
5. **반박 처리** — "그래도 싼 렌즈 끼워주는 거 아니야?" → "적응 안 되면 제가 손해입니다. 어지럽다고 다시 오시면 제가 다 책임져야 하거든요"
6. **3줄 체크박스 정리** — ✔ 테 39,000원 → 렌즈 0원 / ✔ 당근 보고 오시면 안경닦이 서비스
7. **단답 채팅 트리거** — "도수" 한마디만 / "렌즈 0원" 한마디만 / 1분 안에 안내
8. **주소·시간·주차는 맨 마지막**

900~1400자, 한 문장 한 줄 줄바꿈. 모바일 한 호흡에 끊기지 않게.

## 터지는 vs 안 터지는 결정적 차이

**터지는 것 = 원천 콘텐츠(오퍼)를 건드린다.**
- 안경테 39,000원 + 렌즈 0원 (광고가 강한 게 아니라 오퍼가 강하다)
- 깍둑항정살(고기를 깍둑 썰어버림), 한옥 야장(공간 자체)
- 소주병에 1,000원 둘러 그대로 촬영, 행운의 사과(태풍에도 안 떨어짐→수험생)
- 펜션 시설이 구려도 → 고기 무한리필 + 랍스터로 원천 오퍼 갈아끼움 → 전환

**안 터지는 것 = 카피·디자인만 만진다.** "3초 후킹"만 분석하는 콘텐츠, 카피 7개 바꾸기. 원천 그대로 ABO/CBO 돌려도 천장 명확.

체크: 카피 다 지워도 오퍼 한 줄로 클릭할 만한가? 아니면 오퍼부터 다시 짜라.

## 썸네일이 클릭률 올리는 법

1. **AI 깔끔 이미지보다 매장 현장 실사**가 당근 피드에서 더 먹힌다(투박해도 됨).
2. **3단 가격 앵커링 구조** — 상단 ~~200,000원~~ / 중앙 **0원**(가장 크게) / 하단 "안경테만 사면 누진렌즈 무료" / 우상단 뱃지 "3월 31일 마감"
3. **실물 양감** — 렌즈 박스 탑처럼 쌓기, 손바닥 위 렌즈 10알, 계산기 화면에 "103,500" 찍힌 사진
4. **손글씨 가격표 + 진열대** — "테 39,000원~ / 렌즈 0원" 포스트잇
5. **현장 인물 컷** — 안경사 손 클로즈업, 검안 장면(동의 필수)

금지: 보정 과한 스튜디오 컷, 스톡 일러스트, 허위 마감 문구.

## 카피 트리거 실제 사용례

- **숫자 구체**: "압축 3번까지 0원", "1.67 압축렌즈 2만원", "총 70알이 손에 들려서 나가십니다"
- **결핍**: "안경값 반이 렌즈값인 거 아셨나요?", "테는 4만 원인데 총합 15만, 20만…"
- **호기심**: "이거 하나로 종결", "비밀 하나", "어떻게 가능하냐고요?"
- **사회적 증거**: "매일 듣습니다", "다 놀라십니다", "(실제)·(인증)·(후기)" 괄호 표기
- **권위 고유명사**: "케미 1.67 비구면", "쿠퍼비전 1+1", "아큐브 오아시스 원데이", "자이스 ZEISS"
- **장사철학(신뢰)**: "이 동네에서 오래 장사할 사람입니다", "본사 전산이라 저도 어쩔 수 없습니다", "바짓가랑이 잡고 물량 확보해왔습니다"
- **단답 행동유도**: "'도수' 한 마디만", "1분 안에 안내", "당근 보고 왔어요 한마디면 끝"

## 피할 패턴 (실패 신호)

- 추상어 — "혁신적/차별화된/프리미엄/최고의" → 클릭 안 됨
- "후기 쏟아지는 중" 단독(구체 숫자·의심 인용 없이) → 자가발화로 읽힘
- 가격 없이 "가성비/최저가" → 신뢰 깎임
- 페르소나 도식("자신만의 신념을 바탕으로...") → 콘텐츠에 반영 불가, 시간 낭비
- 원천 그대로 두고 카피만 7~8개 바꾸기 → 효율 차이 미미
- "마감임박" 허위 사용 → 신뢰 즉시 손실
- 50대 타겟에 2030 어휘(샤갈, 인스타충) → 읽다 이탈
- 호기심 후크 본문에서 해소 안 함 → 정서적 가치만 주고 전환 0
- 펜션·미용실처럼 원천(공간·시술)이 약한데 광고만 예쁘게 → 전환 안 됨, 원천 오퍼(무한리필·1+1) 갈아끼워야 함

## 메시지 다각화 = 같은 오퍼, 다른 결핍 각도

같은 "렌즈 0원" 오퍼라도 캠페인별로 결핍을 갈아끼운다:
- 2030: "블루라이트 차단" 키워드 직격
- 4050: "도수/난시 페인포인트", "안경값 또"
- 5060: "누진다초점 20만원이 0원", "계산 잘못된 거 아니에요?"

카피를 바꾸는 게 아니라 머릿속에 도는 결핍 문장을 바꾼다. 한 캠페인 안에서 3개 ABO 동시 테스트.
